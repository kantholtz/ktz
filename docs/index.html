KTZ Python Tools#

Welcome to Kantholtz’ Python toolbox. Here, common functionality is gathered which is used throughout my different projects. Although this mostly meant to be my own personal toolkit, maybe you find interesting things to use too. Pull requests, questions etc. are always welcome!

Installation#

Python 3.9 is required.

conda create -n ktz python=3.9
conda activate ktz
pip install ktz

For a local installation with all dev dependencies:

conda create -n ktz python=3.10
conda activate ktz
pip install -r requirements.txt
pip install -e .

Run tests

pytest

Continually run tests

ptw -c

Check code coverage

coverage run -m pytest
coverage report
coverage html

Module Overview#

Collections

  • ktz.collections.buckets() Sort sequences into buckets

  • ktz.collections.unbucket() Unbucket buckets

  • ktz.collections.flat() Flatten collections to desired depth

  • ktz.collections.Incrementer() Auto-assign unique ids

  • ktz.collections.drslv() Resolve string trails in deep dictionaries

  • ktz.collections.dflat() Flatten deep dictionaries

  • ktz.collections.dmerge() Merge deep dictionaries

  • ktz.collections.ryaml() Load and join yaml files

Dataclasses

  • ktz.dataclasses.Index Create an inverted index for dataclasses

  • ktz.dataclasses.Builder Build (immutable) dataclasses incrementally

Filesystem

  • ktz.filesystem.path() Shorthand utility to create pathlib.Paths

  • ktz.filesystem.path_rotate() Keep n previous versions of a file or directory

  • ktz.filesystem.git_hash() Obtain the current git hash

Functools

  • ktz.functools.Cascade Cascade of functions with caching

Multiprocessing

  • ktz.multiprocessing.Actor Process abstraction to be used with a Relay

  • ktz.multiprocessing.Handler Execute code in the Relay’s main process

  • ktz.multiprocessing.Relay Wire Actors together and control the life-cycle

String

  • ktz.string.encode_line() Create a bystestring from a tuple

  • ktz.string.decode_line() Decode a bytestring with custom mappers

  • ktz.string.args_hash() Create a unique and persistent hash from args
