# KTZ - Python Tools

Frequently used and shared functionality across my projects.

## Installation

``` console
$ conda create -n ktz python=3.10
$ conda activate ktz
$ pip install ktz
```

For a local installation with all dev dependencies:

``` console
$ conda create -n ktz python=3.10
$ conda activate ktz
$ pip install -r requirements.txt
$ pip install -e .

# run tests
$ pytest

# to continually run tests
$ ptw -c
```
